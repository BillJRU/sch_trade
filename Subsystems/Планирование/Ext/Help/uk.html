Планування

Призначення підсистеми


Підсистема Планування призначена для планування обсягів продажів, купівель, збирання/розбирання.

Налаштування підсистеми

Планування


Для налаштування розділу Планування в розділі НДІ і адміністрування потрібно виконати наступні налаштування:

  • Для ведення планів продажів потрібно увімкнути функціональну опцію Плани продажів;
  • Для ведення укрупнених планів продажів по категоріях товарів потрібно увімкнути функціональную опцію Плани продажів по категоріях;
  • Для класифікації номенклатури по групам сезонності при плануванні на підставі фактичних даних потрібно увімкнути функціональную опцію Сезонні коефіцієнти, а також встановити періодичність вказання сезонних коєфіцієнтів;
  • Для ведення планів купівель потрібно увімкнути функціональную опцію Плани купівель;
  • Для планування розбирання товарів на комплектуючі та збирання з комплектуючих потрібно увімкнути функціональную опцію Плани збирання (розбирання).

Аналіз даних

Аналіз ключових характеристик можна здійснити за допомогою наступних звітів:

  • Збалансованість планів;
  • Виконання плану продажів;
  • Виконання плану купівель;
  • Виконання плану збирання (розбирання).
